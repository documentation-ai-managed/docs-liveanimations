## Prerequisites

Before starting, ensure you have:

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: Existing Markdown files to import

<Callout kind="info">
  No credit card or installation required. You can start creating documentation spaces immediately after sign-up.
</Callout>

## Create Your Account

Follow these steps to sign up and access the Liveanimations dashboard.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Your Details" icon="user">
    Provide your email, create a password, and verify your account via the email link sent to you.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Answer a few quick questions about your project to personalize your experience.
  </Step>
</Steps>

## Set Up Your First Project Space

Project spaces organize your documentation. Create one in under a minute.

<Steps>
  <Step title="Create New Space" icon="plus">
    From the dashboard, click **New Space** and enter a name like `My Project Docs`.
  </Step>
  <Step title="Configure Basics" icon="settings">
    Set visibility to `Public` or `Private`, and add a description such as "Documentation for my app".
  </Step>
  <Step title="Save and Enter" icon="arrow-right">
    Click **Create** to enter your new space.
  </Step>
</Steps>

## Basic Interface Navigation

The interface includes a sidebar for pages, a preview pane, and an editor.

<Tabs>
  <Tab title="Sidebar" icon="menu">
    Use the left sidebar to manage pages. Drag and drop to reorder.

    - **Pages**: List all documents
    - **Settings**: Space configuration
    - **Integrations**: Connect GitHub or other tools
  </Tab>
  <Tab title="Editor" icon="edit-3">
    The central editor supports Markdown and MDX. Preview updates in real-time.
  </Tab>
  <Tab title="Preview" icon="eye">
    Right pane shows the live rendered documentation.
  </Tab>
</Tabs>

<Expandable title="Advanced Navigation Tips" default-open="false">
  Keyboard shortcuts speed up workflow:

  - `Cmd/Ctrl + K`: Quick search pages
  - `Cmd/Ctrl + S`: Save changes
  - `Cmd/Ctrl + /`: Toggle preview
</Expandable>

## Import Initial Documentation

Upload or paste your existing docs to populate the space.

<CodeGroup tabs="Markdown,API">
  ````markdown
  # Welcome to My Project

  ## Features

  - Real-time collaboration
  - MDX support
  - Custom domains

  ```javascript
  // Example code block
  console.log("Hello, Liveanimations!");
  ```
  ````

  ````javascript
  // API import example
  const response = await fetch('https://api.example.com/v1/spaces/{SPACE_ID}/pages', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'Imported Page',
      content: '# Imported Markdown'
    })
  });
  ````
</CodeGroup>

<Callout kind="tip">
  Use the `https://api.example.com` base URL for all API calls. Replace `{SPACE_ID}` and `YOUR_API_KEY` with your values.
</Callout>
